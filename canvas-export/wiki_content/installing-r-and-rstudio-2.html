We will use R and RStudio to do some basic data analysis during this subject. Below are instructions for installing R and RStudio. Install R before installing R Studio.

R

R is a freely available language and environment for statistical computing and graphics. It is the most common software used in applied behavioural science.

You can download R from the Comprehensive R Archive Network. You can download distributions for Linux, macOS and Windows. Install R before installing R Studio.

You will see a page as follows when you click on that link.

In the box "Download and Install R", click "Download R for MacOS" or "Download R for Windows", depending on your operating system. (If you are using Linux, I trust you can figure this out yourself.)

Downloading for MacOS

If you click "Download R for MacOS" you will see the following:

If you have a newer Mac with M1 or M2 silicon, click the link R-X.X.X-arm64.pkg (X represents the current version numbers). Otherwise, click R-X.X.X-x86_64.pkg. If you're unsure if you have Apple silicon, click the apple in the top left corner of your screen, go to "About this Mac" and the type of chip will be listed.

On clicking that link, the package download will start. Once downloaded, click on the downloaded package and follow the install instructions.

RStudio

RStudio is an integrated development environment (IDE) for R and Python. It provides a console and tools for writing and executing code. 

You can download RStudio from posit. The RStudio Desktop is free.

On following the link, you will see the following.

Click on "DOWNLOAD RSTUDIO". You are then taken to the RStudio Desktop download page.

The page will suggest two steps: first, to download and install R (which you have already done) and second, to download RStudio Desktop. The page will automatically detect your operating system (in my case, MacOS 11+). Click on the "Download RStudio Desktop for XXXX" link, and the file download will commence. Click on the file download and follow the install instructions.
