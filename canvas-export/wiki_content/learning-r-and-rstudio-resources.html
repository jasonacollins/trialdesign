If you have never used R or RStudio before, below I briefly demonstrate the R and RStudio interface and link some introductory resources.

R

If you open R by itself, you are presented with a bare-bones workspace. You can type commands and receive the result.

You can open an editor to work on your code, which provides basic functionality.

However, people typically work within an IDE that makes using R easier. RStudio is one of the most popular for R.

RStudio

The RStudio interface on opening looks as follows. The panel on the left is the R workspace. The panel on the top right shows what objects are stored in memory (the environment). Other tabs in that panel allow you to see your command history, any open connections and a link to a tutorial. The bottom right panel has many purposes: it allows you to find and load files by clicking (rather than the workspace command line), view and load packages, seek help and view visual outputs.

You can also open a new file (typically an R script, R Markdown file or quarto file) where you can develop your code. After selecting File on the top menu, New. File and then R Script, I have the following. I can develop code in the top left panel, and then send it to the bottom left for execution.

Resources

  • RYouWithMe: A gentle, intuitive introduction to using R. Probably the best resource if you are starting from absolute zero. Work through the BasicBasics modules to get a feel of the basic functionality.
  • Getting started with Data in R: A more wordy, slightly less friendly introduction, but a good resource on the basics. Working through Chapter 1 will give you a good grounding in R and RStudio basics.
  • A selection of resources developed in the learnr package.
  • A catalogue of resources provided by RStudio Education gives you a range of suggestions if you want to take this further.
