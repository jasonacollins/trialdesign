Large Language Models (LLM) can support you in code development.

Large Language Models

The quickest way to incorporate a large language model into your code development is to ask a tool such as ChatGPT to produce code.

Please give me R code to determine the required sample size for a test of proportions with 80% power. Assume p1=0.5 and p2=0.6.

The code produced will often have errors or not perform as expected, but it will give you a place to start. If you receive an error or a result that does not make sense, you can ask ChatGPT follow-up questions about the error and how to fix it.

You can also turn to ChatGPT to solve errors in your code. If you cannot determine what is wrong, paste the code into ChatGPT and ask what is wrong with the code.

GitHub CoPilot

GitHub CoPilot is badged as “your coding companion”, providing code suggestions in coding environments.

To access CoPilot, you need to use an IDE such as Visual Studio Code. CoPilot is not available in RStudio.

If you are a student, you can sign up for (free) academic access to CoPilot at this link. You will need a GitHub account to do this.

Visual Studio Code is available here. Download and install.

Install the GitHub CoPilot extension into Visual Studio Code. Instructions on installing the extension are here. On installation, you’ll be prompted to log in to GitHub to gain access to authorise CoPilot.

Once you have done that, you will be given suggestions as you type. If you are typing in an R script, you can type comments (text preceded by #), which are effectively instructions from which CoPilot will produce code.
